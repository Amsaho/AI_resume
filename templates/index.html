About This System

This is a Flask-based Resume Analysis and Job Recommendation System. It helps users upload and analyze resumes, extract skills, and match them with suitable jobs. Key features include:

  • Resume Upload & Parsing (PDF Processing)
  • AI-based Skill Extraction (NLP with spaCy)
  • Job Recommendations Based on ATS Score
  • Face Recognition for Secure Login
  • Admin Panel for Job & User Management
  • PDF Resume Generation with Images
